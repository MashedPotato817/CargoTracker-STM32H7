# 基于STM32H7A3ZI-Q的智能物流追踪系统设计（完整方案）

## 一、项目概述

本项目设计一款基于STM32H7A3ZI-Q的智能货物追踪系统，集成GPS定位、4G通信、NFC激活、温湿度监测与云端控制，实现物流全链路可视化与智能管理。

------------------------------------------------------------------------

## 二、系统核心功能

-   GPS实时定位（ATGM336H）
-   4G云端通信（Air780E）
-   NFC一键激活（PN532 + NTAG215）
-   温湿度监测（SHT31）
-   云端指令控制（MQTT）
-   本地报警（蜂鸣器+LED）
-   断网缓存（W25Q128 Flash）
-   低功耗运行模式

------------------------------------------------------------------------

## 三、硬件BOM清单

### 1. 主控

-   NUCLEO-H7A3ZI-Q（STM32H7A3）

### 2. 通信模块

-   Air780E 4G模块
-   4G PCB/FPC天线
-   SIM物联卡

### 3. 定位模块

-   ATGM336H GPS模块

### 4. NFC模块

-   PN532（I2C）
-   NTAG215 NFC标签（33mm推荐）

### 5. 传感器

-   SHT31温湿度传感器（4Pin）

### 6. 存储

-   W25Q128 SPI Flash

### 7. 电源系统

-   TP4056锂电充电模块（带保护）
-   18650锂电池（3000\~5000mAh）
-   18650电池盒（带开关）
-   MP1584 3.3V DC-DC
-   4.0V DC-DC（4G专用）

------------------------------------------------------------------------

## 四、电源架构

    USB 5V
      ↓
    TP4056（充电+保护）
      ↓
    18650锂电池
      ↓
    ├── 3.3V → STM32 + 传感器
    └── 4.0V → Air780E

------------------------------------------------------------------------

## 五、通信架构

    STM32H7
     ├── USART1 → Air780E（MQTT）
     ├── USART2 → GPS
     ├── I2C1 → SHT31 + PN532 + OLED
     └── SPI1 → W25Q128

------------------------------------------------------------------------

## 六、完整接线表

### Air780E

-   TX → PA10
-   RX → PA9
-   PWRKEY → PB0
-   电源 → 4.0V

### GPS

-   TX → PD5
-   RX → PD6（可选）

### I2C设备（共用PB8/PB9）

-   PN532
-   SHT31
-   OLED（可选）

### SPI Flash

-   CS → PA4
-   SCK → PA5
-   MISO → PA6
-   MOSI → PA7

------------------------------------------------------------------------

## 七、系统状态机

1.  深度睡眠
2.  NFC激活
3.  GPS定位
4.  温湿度采集
5.  数据上传
6.  等待云指令
7.  报警处理
8.  进入休眠

------------------------------------------------------------------------

## 八、软件模块划分

### 驱动层

-   UART / I2C / SPI
-   SHT31驱动
-   PN532驱动
-   Flash驱动

### 通信层

-   Air780E AT指令
-   MQTT协议
-   GPS解析

### 应用层

-   状态机
-   报警逻辑
-   低功耗控制

------------------------------------------------------------------------

## 九、关键设计要点

-   4G模块必须独立供电（4.0V）
-   I2C设备共用总线
-   Flash用于断网缓存
-   NFC用于设备激活
-   必须使用电池保护与DC-DC降压

------------------------------------------------------------------------

## 十、总结

本系统实现物流全链路智能化管理，具备定位、通信、环境监测与云端控制能力，适用于智能物流与冷链运输场景。
