## 🔴 Task 3.2: 数据上报验证（最小闭环）

**负责人：B** | 依赖：Task 2.5 + 3.1 | 预计：2h

只做最核心的一步：设备上电 → MQTT 连接 → 发布一条温湿度 JSON → 云端看到数据。

```json
{"ts":1718123456,"temp":25.5,"hum":60.0,"lat":31.23,"lon":121.47}
```

串口看到 `+CMQTTPUB: 0,0` 且云端控制台显示数据 = 核心功能打通。
